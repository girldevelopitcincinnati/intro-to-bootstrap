Intro to Bootstrap

Learn to make modern, responsive websites using Bootstrap 4!

## Welcome GDI is here to provide affordable and accessible programs to learn software through mentorship and hands-on instruction. **Some "rules":** - We're here for you - Every question is important - Help each other - Have fun!
## Ice Breaker - Who are you? - What do you hope to get out of this workshop? - Do you have any experience with a frontend framework?
## What is Bootstrap? According to Bootstrap, it's the most popular *frontend component library* for developing *responsive*, *mobile-first* projects on the web.
## What is Bootstrap? **Frontend component library** A collection of production-ready HTML/CSS/JavaScript components you can use with little configuration.
## What is Bootstrap? **Responsive** A Web design approach allowing a single webpage to be viewed in response to the size of the screen/web browser it's being viewed with.
## What is Bootstrap? **Mobile-first** Developing a webpage, beginning with your mobile design/layout, and gradually adjusting it so it looks good and behaves well on larger screen sizes.
## What we'll be covering - Setting up your project - The grid - Working with components - Using utility classes - Adding custom CSS
## What we're **NOT** covering - Flexbox - Sass - Bootstrap theming - How to use every Bootstrap feature
## What we're building We're going to build a knock-off Pizza Hut website. This site will combine a bunch of Bootstrap's most commonly used elements in a practical setting. [Demo]()
## What's next? Now that you've built your first Bootstrap project, I encourage you to continue playing around with the framework. Coding sandboxes like [Codepen](http://codepen.io/), [repl.it](http://repl.it/), etc. are great places to experiment. If you're feeling really ambitious, check out [Sass](https://sass-lang.com/) and [flexbox](https://flexbox.io/), or try [building your own Bootstrap theme](https://themes.getbootstrap.com/)!
